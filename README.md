# 批量生成solana 钱包助记词 和地址

功能：

    1、生成一组单助记词多个钱包地址
    2、生成多组助记词和第一个钱包地址


运行时依赖 

        node >= 14.0
        npm list->
            ├── @solana/web3.js@1.41.3
            ├── bip32@3.0.1
            ├── dayjs@1.11.1
            ├── ed25519-hd-key@1.2.0
            ├── lodash@4.17.21
            └── tiny-secp256k1@2.2.1

安装依赖

           npm install

使用方法
    
        node solana3.js [数量] [模式 1:一组助记词 2:多组助记词]

        模式一： 生成一组助记词和 【数量】个钱包地址

        模式二： 生成【数量】组助记词 和 每组助记词的第一个地址


示例模式一、生成一组助记词 和 2个钱包地址
